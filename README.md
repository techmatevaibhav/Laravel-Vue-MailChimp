# Laravel-Vue-MailChimp

- Simple CRUD Laravel Vue Application
- Integration with MailChimp User Existing NewsLetter Laravel Library



## Installation 

- clone repo
- set .env file where you need to set MAILCHIMP_APIKEY and MAILCHIMP_LIST_ID
- `composer install`
- `npm install`
